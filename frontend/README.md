# CURO AI Frontend

A React/Next.js frontend for the CURO AI clinical RAG assistant.

## Features

- Animated chat interface for symptom analysis
- Results dashboard with:
  - Animated text reveals for AI responses
  - List of retrieved medical abstracts
  - Interactive knowledge graph visualization

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Technologies Used

- Next.js 14 with App Router
- React 18
- Tailwind CSS
- React Bits (for animations)
- React Force Graph 2D (for graph visualization)
- Lucide React (for icons)
